# Sulfur CPU Emulator

![Sulfur CPU Logo](docs/image/logo.png)

16ビットRISC風CPUのエミュレータとアセンブラ

## 概要

このプロジェクトは、16ビット固定長命令を持つRISC風CPUのエミュレータとアセンブラの実装です

### 主な特徴

- **16ビット固定長命令**: すべての命令が16ビット（1ワード）
- **ワードアドレス指定**: メモリはワード単位でアドレス指定
- **フォン・ノイマンアーキテクチャ**: 命令とデータが同一メモリ空間に配置
- **8本の汎用レジスタ**: R0-R7（R6=LR、R7=SP）
- **2パスアセンブラ**: 前方参照・後方参照の両方に対応

## アーキテクチャ

### メモリマップ

| アドレス範囲  | 用途                     | サイズ |
| ------------- | ------------------------ | ------ |
| 0x0000-0x1FFF | ROM（ブート領域）        | 8KB    |
| 0x2000-0xEFFF | RAM（プログラム/データ） | 56KB   |
| 0xF000-0xFFFF | I/O領域                  | 4KB    |

### レジスタ

- **汎用レジスタ**: R0-R7（16ビット）
  - R6: リンクレジスタ (LR)
  - R7: スタックポインタ (SP)
- **プログラムカウンタ**: PC（16ビット）
- **フラグレジスタ**: FLAGS（Z, N, C）

### 命令セット

命令セットの詳細は以下を参照。

- [docs/instruction-set.md](docs/instruction-set.md)

READMEには概要のみを記載し、ビット配置・opcode一覧・即値規則・分岐オフセット規則は命令セットドキュメントに分離しています。

## ビルドと実行

### 前提条件

- GCC（C言語コンパイラ）
- Make

### ビルド

```bash
make
```

クリーンビルド：

```bash
make clean && make
```

### 使用方法

現在、プログラムはハードコードされたテストファイルを実行します：

```bash
./cpu_emu
```

#### アセンブリ言語の例

```assembly
# ループのサンプル
loop:
  addi $r1, 10
  beqz $r1, loop
  halt

# 前方参照のサンプル
  addi $r1, 5
  beqz $r1, skip
  addi $r2, 10
skip:
  halt
```

### アセンブラの仕様

- **コメント**: `#` で始まる行はコメント
- **ラベル**: `label_name:` の形式でラベルを定義
- **レジスタ**: `$r0` ～ `$r7` の形式
- **即値**: 10進数で記述
- **出力**: `.s` ファイルを `.mem` ファイルに変換（16進数形式）
